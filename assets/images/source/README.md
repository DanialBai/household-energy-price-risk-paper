# Source Images

Use this folder for original or lightly renamed images imported from related projects.

These files are candidate visual materials, not necessarily final manuscript figures.

## What Belongs Here

- Original maps, plots, diagrams, screenshots, or visual exports from another project.
- Candidate images that may later be selected, redrawn, or converted into manuscript figures.
- Files retained to preserve provenance before editing.

## Naming

Prefer source-aware names:

```text
sourceproject_short-description_original-or-date.ext
```

Example:

```text
westminster-risk_lsoa-income-map_2026-05-23.png
```

## Provenance

For each batch, update this README or add a sidecar note with:

```text
Source project:
Original folder:
Imported on:
Selection rationale:
Reuse/copyright status:
```

Large files over 50 MB should be reviewed before committing. Core files over 100 MB should use Git LFS if they must be versioned.
