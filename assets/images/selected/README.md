# Selected Images

Use this folder for images selected as likely paper assets, presentation materials, or documentation figures.

Files here should be cleaner than `assets/images/source/`, but they are not automatically final manuscript figures.

## What Belongs Here

- Images chosen from the source folder for further use.
- Cleaned or resized versions intended for paper drafting.
- Candidate figures that need final captioning, numbering, or LaTeX integration.

## Naming

Use descriptive names that reflect content rather than version churn:

```text
westminster_lsoa_income_distribution.png
heat-pump_price-risk_scenario-comparison.png
```

Avoid chains such as `final`, `final2`, `v3`, or timestamped debug copies unless comparing versions is the explicit purpose.

## Promotion To Manuscript Figures

Move only confirmed paper figures to:

```text
paper/figures/
```

When a selected image becomes a manuscript figure, record the relationship here or in the figure caption source.
